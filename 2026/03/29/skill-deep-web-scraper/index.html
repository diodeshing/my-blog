AI 技能 #09：深度网页抓取 — 不只是下载 HTML

给一个 URL，AI 帮你把内容提炼成有用的信息。

这个技能能做什么？

  • 抓取文章：提取正文，去除广告和导航
  • 读取文档：长网页/文档页转成结构化内容
  • 提取代码：从教程/文档中提取代码段
  • 系列爬取：自动翻页，抓取系列文章
  • API 文档分析：理解接口结构，提取关键信息

怎么用？

输出

AI 会返回：

  1. 结构化内容：标题、段落、代码块分明
  2. 关键信息提取：重点内容高亮
  3. 代码片段：独立提取，可直接用
  4. 摘要：一篇文章说了什么

适用场景

  • 阅读长文档但没时间
  • 收集某主题的系列文章
  • 学习 API 但文档太长
  • 提取代码示例做参考
