AI 技能 #01：代码助手 — 你说需求，AI 写代码

不是 Copilot 那种补全几行代码，是真正理解需求、写完整模块的能力。

这个技能能做什么？

  • 写代码：描述需求，直接生成可运行的代码
  • 调 Bug：贴错误信息，自动定位并修复
  • 解释代码：看不懂的代码，让它讲清楚
  • Code Review：审查代码质量、安全隐患、性能问题
  • 重构：优化代码结构，不改功能

支持的语言

Python ✅ JavaScript ✅ TypeScript ✅ Shell/Bash ✅ HTML/CSS ✅ SQL ✅

怎么用？

写代码

调 Bug

Code Review

技术原理

AI 通过加载 code-helper/SKILL.md 获取编码规范：

  • 代码风格指南（PEP 8、Airbnb 等）
  • 错误处理最佳实践
  • 安全编码规范
  • 性能优化技巧

生成的代码自动遵循这些规范，不是随便写写。

示例场景

场景 1：数据处理脚本

你说：”读取一个 Excel 文件，按部门汇总工资，输出饼图”

AI 做：

  1. 生成 pandas + matplotlib 代码
  2. 自动处理缺失值
  3. 输出 PNG 图表
  4. 附带使用说明

场景 2：API 接口

你说：”写一个 FastAPI 接口，接收 JSON，存到 SQLite”

AI 做：

  1. 生成完整的 FastAPI 项目结构
  2. 包含数据模型、路由、数据库操作
  3. 附带测试用例

最佳实践

  • ✅ 描述清楚输入输出格式
  • ✅ 说明预期行为和边界情况
  • ✅ 贴完整错误信息（含 traceback）
  • ❌ 不要说”帮我写个程序”太笼统

常见问题

问题 解决
代码跑不起来 贴完整错误信息，AI 会修复
代码能跑但结果不对 描述期望行为，AI 会调整逻辑
想换一种写法 说”用函数式风格重写”等
