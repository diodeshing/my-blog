AI 技能 #05：数据分析 — 丢数据进去，拿洞察出来

不需要会 pandas，丢个 CSV 过来就能得到分析报告。

这个技能能做什么？

  • 统计摘要：均值、中位数、分布、缺失值
  • 趋势分析：时间序列变化、增长率
  • 异常检测：哪些数据点不合群
  • 可视化建议：告诉你该画什么图、怎么画
  • 洞察输出：用文字解释数据说了什么

支持的格式

CSV ✅ JSON ✅ JSONL ✅ 日志文件 ✅ TSV ✅ Excel（需 pandas）✅

怎么用？

输出示例

AI 会给你：

  1. 数据概览：行数、列数、数据类型
  2. 统计摘要：每列的关键指标
  3. 缺失值报告：哪些列有空值
  4. 异常发现：离群点、不合理数据
  5. 可视化建议：推荐图表类型 + Python 代码
  6. 一句话洞察：数据告诉你什么

技术栈

  • pandas — 数据处理
  • matplotlib / seaborn — 可视化
  • json / csv — 格式解析
