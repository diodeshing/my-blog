这个技能能做什么？

  • 查找文件：按名称/类型/日期搜索文件
  • 批量重命名：按规则批量改名
  • 内容搜索：在文件内容中搜索关键词
  • 磁盘分析：找出占空间的大文件
  • 目录整理：自动归类文件
