# AI 技能 #12：博客搭建 — 从零到上线，一条龙

> 你说"我想建个博客"，AI 帮你从零到上线。

## 这个技能能做什么？

- **初始化项目**：安装 Hexo、配置主题
- **主题定制**：Butterfly/NexT 等主题配置
- **部署上线**：自动推送到 GitHub Pages
- **域名绑定**：配置自定义域名
- **SEO 优化**：站点地图、Meta 标签

## 怎么用？

```
"帮我建一个博客"
"把博客部署到 GitHub Pages"
"换个 Butterfly 主题"
"配置一下自定义域名"
```

## 全流程

```
你说"建博客"
  → AI 创建 Hexo 项目
  → 安装主题
  → 配置 _config.yml
  → 初始化 Git
  → 推送到 GitHub
  → 配置 GitHub Pages
  → 上线完成 ✅
```

## 技术栈

- **Hexo** — 静态博客生成器
- **Butterfly** — 主题
- **GitHub Pages** — 免费托管
- **Git** — 版本控制和部署
