# AI 技能 #16：通知推送 — 自动化消息，定时触达

> 定时查天气推送到手机？自动发邮件汇报？一个技能搞定。

## 这个技能能做什么？

- **发邮件**：通过 SMTP 发送邮件
- **Webhook 触发**：调用任意 Webhook URL
- **Telegram 推送**：发消息到 Telegram
- **定时任务**：设置定时执行
- **资讯聚合**：定时抓取并推送新闻

## 怎么用？

```
"帮我发一封邮件给 xxx@xxx.com"
"触发这个 Webhook: https://..."
"每天早上 8 点推天气预报到我邮箱"
"每周五发周报"
```

## 支持的渠道

| 渠道 | 状态 |
|------|------|
| Email (SMTP) | ✅ |
| Webhook | ✅ |
| Telegram | ✅ |
| 钉钉 | 🔧 可扩展 |
| 企业微信 | 🔧 可扩展 |
