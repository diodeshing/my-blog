# AI 技能 #14：每日总结 — 自动整理你今天干了啥

> 下班时说"写个总结"，AI 帮你整理今天做了什么。

## 这个技能能做什么？

- **自动回顾**：分析今天的对话和操作记录
- **生成日报**：结构化的工作总结
- **写成博客**：自动发到博客上
- **发邮件**：通过邮件推送日报
- **存档保存**：保存到 memory 文件

## 怎么用？

```
"帮我写今天的总结"
"写个日报"
"今天做了什么，整理一下"
```

## 输出格式

```
## 今日工作

### 完成事项
- 搭建了 Home Assistant 技能
- 安装了 Whisper 语音转文字
- 创建了 5 篇博客文章

### 进行中
- API 测试器待完善

### 明日计划
- 完善剩余技能文档
- 部署新博客文章
```

## 全流程

```
你说"写今天的总结"
  → AI 回顾今天所有对话
  → 提取关键任务和成果
  → 生成结构化日报
  → 自动发成博客（可选）
  → 发邮件（可选）
```
