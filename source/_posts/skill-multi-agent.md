# AI 技能 #18：多智能体协作 — 复杂任务，分而治之

> 一个人干不了的活，派一个小队去干。

## 这个技能能做什么？

- **任务拆分**：大任务拆成多个子任务
- **并行执行**：多个 Agent 同时干活
- **结果汇总**：收集各 Agent 的输出，整合结果
- **不同模型**：不同子任务用不同 AI 模型
- **隔离执行**：每个 Agent 在独立环境中运行

## 适用场景

| 场景 | 为什么需要多 Agent |
|------|-------------------|
| 大型代码重构 | 一个人改太慢，多人并行 |
| 多文件分析 | 同时读多个文件，各自分析 |
| A/B 对比测试 | 两个 Agent 用不同策略执行 |
| 研究 + 写作 | 一个搜资料，一个写文章 |

## 怎么用？

```
"帮我重构这个项目，每个模块一个 Agent"
"同时分析这 5 个日志文件"
"一个 Agent 搜资料，另一个 Agent 写报告"
```
