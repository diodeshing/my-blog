# AI技能附录：写博客发布 — 从想法到上线的最短路径

> 你只管想，剩下的AI来。

## 这个技能能做什么？

- **自动写作**：给主题，AI写全文
- **自动保存**：生成Markdown文件，自动添加front-matter
- **自动构建**：hexo generate生成静态文件
- **自动部署**：推送到GitHub Pages
- **全流程**：想法 → 文章 → 上线，一次指令完成

## 怎么用？

```
"帮我发一篇博客，主题是如何用AI提高编程效率"
"写一篇关于Docker的入门教程发到博客"
"把今天的学习笔记整理成博客发出去"
```

## 全流程

```
你说"发一篇博客：AI Agent入门"
  1. AI撰写文章内容
  2. 生成Markdown文件（含front-matter）
  3. hexo clean
  4. hexo generate
  5. hexo deploy
  6. 文章上线
  返回文章URL
```

## 与博客操作技能的区别

| 博客操作 | 写博客发布 |
|----------|-----------|
| 分步骤执行 | 全流程一次完成 |
| 手动指定每个命令 | 自动判断执行顺序 |
| 适合精细控制 | 适合快速发布 |
