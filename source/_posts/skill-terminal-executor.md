# AI 技能 #03：终端执行器 — 带安全网的命令行

> 不是让它随便跑 `rm -rf`，是有分级权限的安全执行框架。

## 这个技能能做什么？

- 执行系统命令（`pip install`、`git commit`、`dir` 等）
- 自动分级：只读免审，危险拦截
- 越界报备：访问工作区外的目录需要说明原因
- 超时保护：命令超过 60 秒自动终止
- 完整日志：谁在什么时候跑了什么命令

## 权限分级

| 级别 | 说明 | 示例 |
|------|------|------|
| 🟢 Level 0 | 免审只读 | `dir`、`cat`、`python --version` |
| 🟡 Level 1 | 工作区内操作 | `pip install`、`git commit` |
| 🟠 Level 2 | 越界需报备 | 访问 `D:\`、全局安装 |
| 🔴 Level 3 | 危险拦截 | `rm -rf`、`format`、`del *.*` |

## 怎么用？

直接说人话：

```
"帮我看一下 D:\ 目录有哪些大文件"
"pip install pandas"
"看看 git status"
"清理一下 temp 目录"
```

AI 会判断权限等级，安全的直接跑，危险的先问你。

## 安全机制

- **工作区隔离**：默认只能在 workspace 内操作
- **白名单模式**：只有明确授权的命令才会执行
- **超时保护**：防止卡死
- **日志审计**：所有命令可追溯
