# AI 技能 #10：网络研究 — AI 帮你做调研

> 你说主题，它搜多个来源，对比分析，给你一份研究报告。

## 这个技能能做什么？

- **多源搜索**：同时搜索多个信息来源
- **对比分析**：不同来源的信息交叉验证
- **报告生成**：结构化的研究报告
- **事实核查**：验证信息的准确性
- **趋势分析**：追踪某话题的发展趋势

## 怎么用？

```
"帮我研究一下 2026 年 AI Agent 的发展趋势"
"对比一下 React 和 Vue 的最新版本差异"
"调研一下目前市面上最好的 Markdown 编辑器"
"Rust 和 Go 在系统编程方面哪个更好？"
```

## 输出格式

1. **背景概述**：这个话题是什么
2. **关键发现**：最重要的 3-5 个结论
3. **详细分析**：每个结论的支撑证据
4. **对比表格**：多方案对比（如适用）
5. **建议**：基于研究给出建议
6. **来源列表**：引用了哪些资料

## 与直接搜索的区别

| 普通搜索 | AI 研究 |
|----------|---------|
| 返回链接列表 | 返回分析报告 |
| 你需要自己读 | AI 帮你提炼 |
| 一个来源 | 多来源对比 |
| 只有信息 | 有结论和建议 |
