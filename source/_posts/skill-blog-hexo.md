# AI 技能 #13：博客操作 — 写文章、发文章，一键完成

> "发一篇博客" → AI 写完 → 自动部署上线。

## 这个技能能做什么？

- **创建文章**：`hexo new` 自动生成 Markdown 模板
- **写内容**：你给主题，AI 写全文
- **本地预览**：启动本地服务器预览
- **部署发布**：`hexo clean && hexo deploy` 一键上线
- **管理标签/分类**：自动组织文章元数据

## 怎么用？

### 写一篇新文章

```
"帮我发一篇博客，主题是 Docker 入门"
"写一篇关于 2026 年编程趋势的文章"
```

### 部署

```
"把博客部署上去"
"部署一下新文章"
```

### 管理

```
"看看博客里有哪些文章"
"本地预览一下"
```

## 全流程

```
你说"发一篇博客：Docker 入门"
  → AI 创建 Markdown 文件
  → AI 撰写内容（标题、正文、代码示例）
  → 自动添加 front-matter（日期、标签、分类）
  → hexo generate
  → hexo deploy
  → 文章上线 ✅
```
