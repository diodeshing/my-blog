# AI 技能 #19：文件管理 — 智能整理你的文件

> 桌面一团乱？AI 帮你整理。

## 这个技能能做什么？

- **查找文件**：按名称/类型/日期搜索文件
- **批量重命名**：按规则批量改名
- **内容搜索**：在文件内容中搜索关键词
- **磁盘分析**：找出占空间的大文件
- **目录整理**：自动归类文件

## 怎么用？

```
"帮我找一下 D 盘里所有的 .log 文件"
"把这些文件批量重命名为 date_name 格式"
"搜索包含'error'的所有文件"
"看看 D 盘哪些文件夹最大"
"整理一下下载目录"
```

## 安全机制

- 删除操作使用回收站，可恢复
- 批量操作前会先预览
- 不动系统文件
