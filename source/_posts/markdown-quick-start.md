# Markdown 速成：10 分钟掌握写作必备语法

Markdown 是程序员的"写作语言"。学一次，写博客、文档、README 全都能用。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式

```markdown
**加粗**
*斜体*
~~删除线~~
`行内代码`
> 引用
```

## 列表

```markdown
无序列表：
- 第一项
- 第二项

有序列表：
1. 第一步
2. 第二步

任务列表：
- [x] 已完成
- [ ] 待完成
```

## 代码块

````markdown
```python
def hello():
    print("Hello World")
```
````

常用语言标记：`python` `javascript` `bash` `json` `yaml`

## 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](https://example.com/img.png)
```

## 表格

```markdown
| 姓名 | 年龄 | 职业 |
|------|------|------|
| 张三 | 25   | 开发 |
| 李四 | 30   | 设计 |
```

## 分割线

```markdown
---
```

## 转义字符

```markdown
\* 不会变斜体
\# 不会变标题
```

## 快捷操作（VS Code）

| 操作 | 快捷键 |
|------|--------|
| 加粗 | Ctrl+B |
| 斜体 | Ctrl+I |
| 预览 | Ctrl+Shift+V |

## 常见坑

1. **换行**：行尾加两个空格，或者空一行
2. **缩进**：代码块内缩进用空格，不要用 Tab
3. **特殊字符**：`*` `#` `>` 等有特殊含义，需要时加 `\` 转义

## 总结

> Markdown 的精髓是"内容优先，格式其次"。你不需要记住所有语法，常用的就那 10 个，写多了自然就记住了。
