# AI 技能 #11：天气查询 — 秒回天气预报

> "北京明天天气怎么样？" 秒回。

## 这个技能能做什么？

- **当前天气**：温度、湿度、风速、体感温度
- **天气预报**：未来 3-7 天
- **全球覆盖**：任意城市都行
- **无需 API Key**：直接用

## 怎么用？

```
"北京今天天气怎么样？"
"上海未来三天天气"
"深圳明天要不要带伞？"
"New York weather tomorrow"
```

## 输出

- 🌡️ 温度（最高/最低/当前）
- 💧 湿度
- 🌬️ 风速风向
- ☀️ 天气状况
- 🌧️ 降水概率
